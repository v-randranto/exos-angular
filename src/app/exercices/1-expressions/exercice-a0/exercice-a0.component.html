Commandes CLI

npm install -g @angular/cli
=> installe le CLI d'Angular (ai fait l'install avec css au lieu de scsscomme demandé, pas grave)

ng new exercices
=> crée un projet 'exercices'

ng serve
=> démarre le serveur

ng generate component exercices/1-expressions/exercice-a0 // raccourci ng n c
=>crée un composant nommé `ExerciceA0Component` dans le dossier `src/app/exercices/1-expressions`

ng generate service mon-dossier/todo-service // raccourci ng g s
=>crée un service nommé `TodoService` dans le dossier `src/app/mon-dossier`

ng generate module CustomerDashboard
=>crée un dossier nommé `customer-dashboard` sous `src/app/` contenant un fichier `customer-dashboard.module.ts`
